其他相关的知识

  1. 为文章页面最下面的tags添加展示图标 # hexo ，在主题配置文件中更改 tag_icon: true 。或者打开 themes/next/layout/_macro/post.swig 文件,搜索 rel="tag"># ,将那个 # 换成 <i class="fa fa-tag"></i> 即可显示图标

  2. 修改Hexo默认的端口号，既可以 hexo server -p 80 时指定，也可修改 www/node_modules/hexo-server/index.js 中默认的 port: 4000
