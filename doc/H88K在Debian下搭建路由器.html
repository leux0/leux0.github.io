H88K在Debian下搭建路由器

发表于 2024-03-25 | 分类于 Linux

前言说明

  1. 硬件设备用的是：Hinlink H88K V3，无线网卡：MT7916，移动模组：RM500Q-GL，NVME硬盘：Intel 760P

  2. 系统用的是 自编译修改版Armbian 24.5 ，backport 内核支持 MT7916 网卡。RM500Q开启支持 QMI ECM MBIM RNDIS 等驱动，当前教程使用的是 ECM 模式

  3. 网口规划：变压器故障的那个 eth0 usb0 wlan0 做 WAN，其余网口做 LAN

准备工作

  1. Linux下默认的网卡名称是随机的，为了方便后续操作先禁止网卡重命名

sudo bash -c "echo 'extraboardargs=net.ifnames=0' >> /boot/armbianEnv.txt"

  1. 开启IPv4内核转发

sudo bash -c "echo 'net.ipv4.ip_forward=1' >> /etc/sysctl.conf"

  1. 安装所需软件

sudo apt install dnsmasq hostapd bridge-utils

  1. 由于我使用的是带GNOME桌面的版本，所以需要让NetworkManager不管理上面作为LAN的网卡
