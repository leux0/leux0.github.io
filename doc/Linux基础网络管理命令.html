Bluetooth

  1. 安装Bluetooth连接管理程序：sudo apt install bluez

  2. 添加pi用户到蓝牙组后重启，否则普通用户pi无法操作蓝牙：sudo usermod -G bluetooth -a pi

  3. 如果不添加用户到蓝牙组，那么全局都需要以root权限运行

  4. 重启后蓝牙连接不需要重复配对，只需要执行：connect F4:4E:FD:F0:2A:69 即可
