Github加速之修改hosts

发表于 2020-06-06 分类于 Other

 

1 获取GitHub官方CDN地址

打开 https://www.ipaddress.com/ 查询以下几个网址的IP地址

github.com
assets-cdn.github.com
github.global.ssl.fastly.net

修改系统Hosts文件

添加如下到系统hosts文件(需管理员权限)：C:\Windows\System32\drivers\etc\hosts

刷新系统DNS缓存

以管理员身份打开系统命令行执行 ipconfig /flushdns 手动刷新系统DNS缓存。
